# Project Overview

Credence Analytics is a data analytics project that investigates how transaction behaviour can be used to predict financial reliability among individuals who lack traditional credit histories.

This project supports the broader vision of **Credence**, an AI-powered Digital Financial Identity platform designed to help financial institutions gain additional, consent-based insights into customer financial behaviour.

The project follows the Cisco Data Analytics Lifecycle and uses publicly available datasets to simulate real-world financial analysis.

---

# Business Problem

Millions of financially active individuals use digital financial services every day.

Despite this, many remain underserved because they lack traditional credit histories or formal proof of income.

Financial institutions therefore have limited information when assessing these customers.

This project explores whether transaction behaviour can provide meaningful indicators of financial reliability.

---

# Business Question

> **How can transaction behaviour be used to predict financial reliability among individuals who lack traditional credit histories?**
